# cmdchallenge solutions 

My personal solutions to challenges from [cmdchallenge.com](https://cmdchallenge.com) — a site with fun one-liner shell challenges.

## Structure

```
<number_of_the_challenge>_<name_of_the_challenge>
    challenge
         ...
    solutions
         ...     
```

## How it works

Each challenge has its own folder with the challenge-related files and possible solutions.
